Getting Started

KorvaqScrip is a beginner-friendly programming language with a straightforward syntax, ideal for clear and concise coding. It uses the ".kq" file extension and features simple constructs and data types.

Basic Syntax

Statements are terminated with a semicolon (;), and code blocks are enclosed in braces ({}).

Variable Declarations

  • Mutable Variables: Declared with let, allowing reassignment (e.g., let x = 5;).
  • Immutable Variables: Declared with make, these cannot be changed (e.g., make y = 3;).
  • Deleting Variables: Use delvar to remove variables and prevent memory leaks (e.g., delvar x;). You can delete all variables with delvar all;.

Data Types

  • Numbers: Support mathematical operations (e.g., let sum = 5 + 10;).
  • Strings: Defined using double quotes (e.g., let greeting = "Hello, World!";).
  • Booleans: Can be true or false (e.g., let isActive = true;).

Control Structures

  • Conditional Statements: Use if and else for conditional logic (e.g., if (x > 5) { ... }).
  • Loops: loop iterates over a range, while while iterates based on a condition (e.g., while (example == false) { ... }).

Functions

Declared with func, they accept parameters and return values (e.g., func addVal(a, b) { ... }).

Output

Use show to print values to the console (e.g., show "Hello, World!";).

Expressions

Support assignments and binary expressions for arithmetic and logical operations.

Arrays

Defined with square brackets ([]) to hold multiple values (e.g., let numbers = [1, 2, 3];). You can add values using arrayadd, get the length with arraylength, and access elements with arrayName[index].

Comments

Single-line comments can be added using //.

Linking Files

Use connect to link different .kq files together.

Running Asynchronous

Use async {} for asynchronous execution.

File Management

Read files using the read keyword (available in the KorvaqShell application).

Examples

  • Check if a number is even or odd.
  • Calculate the sum of two numbers.
  • Loop through an array of fruits.

Recommended Approaches

  • Use let for variable declarations.
  • Manage memory effectively with delvar and delvar all; to avoid leaks.

KorvaqScrip is designed to simplify coding while providing essential programming capabilities for effective software development.
